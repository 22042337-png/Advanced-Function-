Parent function: |x|

g(x) = -\(\frac{3}{4}\)\(\frac{12}{11}|x + \frac{235}{100}|\) + 1.45

Transformation:

The reason why I chose the absolute function for this graph because the roof of the castle is close to the shape of absolute function. This equation evolves the absolute function. The discrepancies from the parent function are:

  • Reflection on x-axis
  • Vertical compression by factor of 3/4
  • Horizontal compression by factor of 11/12
  • Vertical translation upward for 1.45 units
  • Horizontal translation toward left for 2.35 units

Domain and Range

The domain of this function is {−3 ≤ x ≤ 3.1}, the range of this function is {0.69 ≤ y ≤ 1.5}.

Thinking

The most challenging part is connecting different functions together. I needed to allocate the different functions to their position, and found the focal points. Then, I used the domain to present a part of them. The effect of my product shows as if every line is connected together.
