Parent function: √x

1.6√(4.6 - x) − 4.5 {−1 ≤ x}

Transformation:

The reason why I chose this function is that the kink structures are easily found in slides. This equation evolves the root function. The discrepancies from the parent function are:

  • Reflection on y-axis
  • Vertical stretch by a factor of 1.6
  • Horizontal translation toward the right for 4.6 units
  • Vertical translation downward for 4.5 units

Domain and Range

The domain of this function is: {−4.2 ≤ x ≤ 4.6}, the range of this function is: {−4.5 ≤ y ≤ 2.5}.

Thinking

The most challenging I met while I was working on the root function project was finding the appropriate functions to fit the shape of the slide. As the shape of this slide is a helix, the most functions that I had used were reciprocal function, log function, and root function; those functions can create the twisted line to fit the slide shape.
